# Pursuit and Distractions Organizer (Proof of concept)

Organize your random thoughts and wants into more accurate reasoning. Saves into json.

## How to run
if you have python, you could run the server by doing `python -m http.server` or just run `make`

but if you don't need, just simply open the html file in your browser and you are good to go.
